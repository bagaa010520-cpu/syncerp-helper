Ирц бүртгэлийн мэдээлэл хэрхэн оруулах вэ?

Ирц бүртгэлийн мэдээлэл оруулахдаа хянах самбараас Цаг бүртгэл >> Ирцийн файл оруулах >> ИРЦИЙН ФАЙЛ ОРУУЛАХ цэс рүү орох бөгөөд ажилчдын цаг бүртгэлийн мэдээллийг төхөөрөмжөөс татан авна. Мөн ENVISION ФАЙЛ ОРУУЛАХ боломжтой бөгөөд үүний тулд урьдчилан .CSV өргөтгөлтэй файл бэлдсэн байх шаардлагатай.

Ирц бүртгэлийн файлын жагсаалт




1.ШИНЭ ФАЙЛ ОРУУЛАХ - Бэлдсэн .CSV өргөтгөлтэй файлаа системд оруулна.





Төхөөрөмжийн нэршил болон файлын төрлөө сонгосны дараа МЭДЭЭЛЭЛ хэсэг дээр тайлбар гарч ирэхийг анхаарна уу! Үүний дараа файлын зам хэсэгт CHOOSE FILE товчийг даран файлаа оруулна. Бүх мэдээллийг бөглөсний дараа ХАДГАЛАХ товчийг дарснаар үндсэн хэсэг дээр файлын мэдээлэл гарч ирнэ. Энэхүү хэсэгт ЗАСВАРЛАХ боломж байхгүйг анхаарна уу! 2.Бичлэгийн тоо - Файл дахь мөрийн тоог илэрхийлэх бөгөөд Бичлэгийн тоон дээр дарснаар файлын агуулга болон дэлгэрэнгүй мэдээллийн хэсэг нээгдэнэ.



Нэмэлт тайлбар:
Ажилтны дугаар талбар дахь файл болон систем-ийн дугаар нь ажилтны системд бүртгэгдсэн, мөн тухайн файлд байрлаж буй дугаарыг илэрхийлнэ.

3. ШИНЭ ENVISION ФАЙЛ ОРУУЛАХ - Бэлдсэн .CSV өргөтгөлтэй файлаа системд оруулна.


Файлын зам хэсэгт CHOOSE FILE товчийг даран файлаа оруулна. Файлаа оруулсны дараа ХАДГАЛАХ товчийг дарснаар үндсэн хэсэг дээр файлын мэдээлэл гарч ирнэ. Энэхүү хэсэгт ЗАСВАРЛАХ боломж байхгүйг анхаарна уу!
4. БИЧЛЭГИЙН ТОО - Файл дахь мөрийн тоог илэрхийлэх бөгөөд Бичлэгийн тоон дээр дарснаар файлын агуулга болон дэлгэрэнгүй мэдээллийн хэсэг нээгдэнэ.
